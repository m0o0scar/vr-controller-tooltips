# Vite React+TS Template

## Install

```
npx degit m0o0scar/vite-react-ts#vr my-project
cd my-project

npm i
npm run dev
```

For more info regarding starting a project with community template, please see [here](https://vitejs.dev/guide/#community-templates).

## Publish

```
npm run build:npm
npm publish
```
